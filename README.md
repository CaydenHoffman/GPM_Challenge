# GPM_Challenge
This project is a React web application that fetches and displays agronomy articles from the Great Plains Ag public API. Users can browse articles, filter by tag, and search articles by keyword. The layout and functionality are modeled after the Agronomy page on the Great Plains Ag website.

---

## Technologies Used

- React
- Vite
- JavaScript 
- CSS
- Fetch API

---

## Getting Started
follow these steps to run locally

1. clone the repo
2. navigate to directory 
3. if you dont already have dependencies installed: npm install
4. run dev server: npm run dev
5. open url provided by vite. usually localhost:5173
